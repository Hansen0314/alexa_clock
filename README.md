# alexa_clock
时钟在日常的生活中是必不可少的一个组件，随着科技的发展，我们是不是也应该将常规的时钟改造成充满硬核科技的超级时钟呢？我们可以将alexa服务内置在我们的时钟里面，重而使时钟不仅能看时间还能查询天气收听新闻等等。
